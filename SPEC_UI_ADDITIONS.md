## 1. Modal משימה

### מצב נוכחי

```
שם המשימה *
קורס
תאריך פתיחה | תאריך יעד
תיאור
הסטטוס שלי (4 כפתורים)
נוסף ע"י
[שמור] [מחק]
```

### מה חסר — עדיפות A

#### 1.1 שדה: עדיפות (Priority)

```
┌────────────────────────────────────────┐
│  עדיפות                                │
│  [🟢 נמוך]  [🟡 בינוני ●]  [🔴 גבוה]  │
└────────────────────────────────────────┘
```

**שמירה ב-Firebase:** `priority: "low" | "medium" | "high"` (ברירת מחדל: `"medium"`)
**תצוגה בכרטיסייה:** dot צבעוני בפינה + מיון — גבוה ראשון בעמודה

---

#### 1.2 שדה: סוג הגשה

```
┌────────────────────────────────────────────────────┐
│  סוג הגשה                                          │
│  [📧 מייל]  [📤 Dropbox]  [🖨 ידנית]  [💻 Moodle]  │
│  [✏ אחר: _______________]                           │
└────────────────────────────────────────────────────┘
```

**שמירה:** `submission_type: "email" | "dropbox" | "physical" | "moodle" | "other"`
**other_text:** `submission_note: string`

---

#### 1.3 שדה: קישור להגשה

```
┌────────────────────────────────────────────────────┐
│  קישור להגשה (אופציונלי)                           │
│  [🔗 https://...                              ] 📋  │
└────────────────────────────────────────────────────┘
```

**שמירה:** `submission_url: string | null`
**תצוגה בכרטיסייה:** כפתור `[🔗 הגש]` אם יש URL

---

#### 1.4 שדה: תת-משימות (Checklist)

```
┌────────────────────────────────────────────────────┐
│  רשימת משנה               [+ הוסף סעיף]           │
│  ☐  טיוטה ראשונה                           [🗑]   │
│  ☑  קריאת חומר בסיסי                       [🗑]   │
│  ☐  עריכה סופית                            [🗑]   │
│                                                    │
│  ████████░░  2/3 הושלמו                           │
└────────────────────────────────────────────────────┘
```

**שמירה:**
```javascript
subtasks: [
  { id: "s1", text: "טיוטה ראשונה", done: false },
  { id: "s2", text: "קריאת חומר בסיסי", done: true },
  { id: "s3", text: "עריכה סופית", done: false }
]
```

**תצוגה בכרטיסייה Kanban:** `☑ 2/3` progress mini-bar

---

#### 1.5 שדה: קובץ מצורף / לינק חומר

```
┌────────────────────────────────────────────────────┐
│  חומרים / קישורים קשורים          [+ הוסף לינק]   │
│  📄 חומר קריאה.pdf              [🗑]              │
│  🔗 Dropbox — תיקיית פרויקט     [🗑]              │
└────────────────────────────────────────────────────┘
```

**שמירה:**
```javascript
attachments: [
  { label: "חומר קריאה", url: "https://...", type: "pdf" },
  { label: "Dropbox — תיקיית פרויקט", url: "https://...", type: "link" }
]
```

---

#### 1.6 שדה: קבוצה / שותפים

```
┌────────────────────────────────────────────────────┐
│  קבוצה                                             │
│  [👤 עבודה יחידנית ●]  [👥 עבודה קבוצתית]         │
│                                                    │
│  שותפים (אופציונלי)                                │
│  [🏷 הוסף שם...]      דנה ×   יוסי ×              │
└────────────────────────────────────────────────────┘
```

**שמירה:**
```javascript
is_group_work: boolean,
partners: string[]
```

---

#### 1.7 שדה: הערות פרטיות (visible only to me)

```
┌────────────────────────────────────────────────────┐
│  📝 הערות אישיות  (נשמר לוקלית, לא ב-Firebase)     │
│  [_____________________________________________]   │
│  [_____________________________________________]   │
└────────────────────────────────────────────────────┘
```

**שמירה:** `localStorage['task_private_notes'][taskId]`

---

#### 1.8 כפתור: העתק משימה

בheader של ה-modal (ב-edit mode):

```
┌─────────────────────────────────────────┐
│  ✏ ערוך משימה          [📋 שכפל] [✕]   │
└─────────────────────────────────────────┘
```

---

#### 1.9 כפתור: הזכר לי (Reminder)

```
┌────────────────────────────────────────────────────┐
│  🔔 תזכורת                                         │
│  [יום לפני ●]  [3 שעות לפני]  [שעה לפני]  [ללא]  │
└────────────────────────────────────────────────────┘
```

**מימוש:** localStorage + `setTimeout` / Service Worker בעתיד

## 4. Kanban Card Actions

### מצב נוכחי

```
כרטיסייה לחיצה → פתיחת modal עריכה
```

### מה חסר — עדיפות A

#### 4.1 Context Menu / Quick Actions (בhover)

```
┌──────────────────────────────────────────────────┐
│  🔴 דחוף    🟣 תחקיר                             │
│                                                   │
│  ביקורת מחזה — "אוטלו"                           │
│                                                   │
│  📅 20/03   🎓 תחקיר #7                          │
│                                                   │
│  ══════════════════════ ← מוצג בhover            │
│  [→ קדם]  [✏ ערוך]  [📋 שכפל]  [🗑 מחק]         │
└──────────────────────────────────────────────────┘
```

---

#### 4.2 כפתור "→ קדם" — Quick Status Advance

```javascript
// קדם לסטטוס הבא:
none → todo → inprogress → submitted
```

לחיצה על "→ קדם":
- מקדם את הסטטוס הפרטי של המשתמש
- הכרטיסייה זזה לעמודה הבאה עם אנימציה
- Toast: `✓ הועבר ל"בתהליך"`

---

#### 4.3 Drag & Drop (עדיפות B)

- Drag כרטיסייה → Drop לעמודה אחרת
- בזמן drag: עמודות מודגשות עם border כחול
- Drop → שמירה מיידית לFirebase

---

#### 4.4 Multi-Select (עדיפות B)

- Long press / Ctrl+Click → מצב בחירה
- Checkboxes מוצגים על כל card
- Action bar בתחתית:

```
[☑ 3 נבחרו]  [→ קדם הכל]  [🗑 מחק הכל]  [📚 שנה קורס]  [✕ בטל]
```

## 6. לוח שנה

```
[פאנל יומן]
  ← [תאריך] →  [+ אירוע]

    [שיעורי ICS]
    [משימות עם due/opening]
```

### מה חסר — עדיפות A

```
┌──── ראשון 15.3 ───[+]──┐

```
